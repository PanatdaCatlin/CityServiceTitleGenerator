City and Service Title Generator

By Panatda Catlin

Welcome, this is just a simple application where you can enter a service type and name of any city and it will print out in a list of the service plus name and sort it alphabet order according to the city.


This is super easy to use:

  • Enter each service spearate each service with a comma
  • Enter each city name spearate each service with a comma
  • Hit Submit when you are done
  • List should print
  • Hit Clear to try again
